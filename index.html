HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element defines a section in the document.
  • The H1-H6 elements represent heading types and weight.
  • The ul, ol and li elements represent unordered and ordered lists.

CSS

  • Inline style CSS applies to specific HTML tags.
  • Most common way to style a webpage is by using an external CSS style sheet.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Text-align is a way to center a title or heading on a webpage.
  • The display property is assigned with the block value, which assigns each image to start on a new line.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds code to main branch
  • git commit -m: commits code to main branch
  • git pull origin branch-name: pulls code from remote branch to working branch
  • git push origin branch-name: pushes changes from working branch to remote branch
  • git checkout branch-name: switches code to different branch

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is a way of introducing decision-making into our code. Code will only execute if the statement meets a condition, or is true.
  • We can have more than one condition by using else if statement.
  • An array is a single variable that is used to hold a group of data.
  • Arrays are typically used to hold data that is related in some way.
  • Arrays are especially useful when we want to perform a task on multiple pieces of data at once.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • To declare a function, we use the function keyword and then give our function a unique name followed by parentheses.
  • Then we add curly brackets {} that contain the code for the task we want to perform.
  • To call a function, we use the name of the function in our JavaScript file where we want the code to execute.
